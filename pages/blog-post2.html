What is Python?

October 2022

Python is an easy-to-learn and easy-to-read object-oriented, high-level programming language with dynamic semantics.[ref](#whatispython). Dynamic semantics means that its variables can change datatypes and don't need to be statically defined. Eg: 

          
int1 = 10

int1 = int1 / 5

print(type(int1))
>>> <class 'float'>

Because Python syntax is easy to read it is great for coding beginners - the code can almost be read in plain English or pseudo-code. Also the code does more with less, it has abstracted away many processes so that code can be shorter and faster to write. Eg: 

          
  for i in range(8):
    print(i)
  >>> 0 1 2 3 4 5 6 7
