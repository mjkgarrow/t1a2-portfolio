Data types

October 2022

In programming languages, data are stored in variables. The kind of data a variable can store is called its data type. A data type dictates what kinds of values can be stored and what kinds of operations can be done on that data. Let's look a few examples of data types.


Here is a 'str' data type - they are surrounded by single or double quotation marks. 'str' is short for 'string', ie: a string of characters. 'str' contain text data (which may include numbers, which shouldn't be confused with ints). 'str' can be printed as a literal - meaning it will print what is in the quote marks. 

          
  text = "Hello"
  print(text)
  >>> Hello

'str' are iterable, you can loop through each character in a string. 

          
  print(text[1])
  >>> e

'str' can be concatenated, which means 'put together in a series'. 

          
  text1 = "Ma"
  text2 = "tt"
  print(text1 + text2)
  >>> e

'Int' data type - they are numeric and a whole number, either positive or negative. 'Int' is short of 'integer'. Ints can do mathematical operations, eg: addition, subtraction, multiplication 

          
  num1 = 5
  num2 = 3
  num3 = 12351348619348234754
  num4 = -252345
  print(num1 + num2)
  >>> 8
