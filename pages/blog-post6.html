Lists in Python

October 2022

A list is a mutable and ordered collection of data, known as a compound data type. Lists might contain items of different types, they can be indexed and sliced, and they are iterable. They are represented by square brackets and elements separated by commas, eg: 

          
  my_list = [1, 2, 3, 4]

Lists are perfect for storing data that needs to be appended, removed, changed or iterated over. Because they are ordered, each element in a list remains in position, which means you can index into them consistently. An example of using a list is when you want to make a collection of all the names in a phonebook: 

          
  names_list = ['Matt', 'Beth', 'Rohan', 'Anna', 'Matt']

Lists include a number of built-in methods such as len() (finds the length of the list), append() (adds to the list), pop() (removes from the list).
